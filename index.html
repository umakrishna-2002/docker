Hello from Jenkins CI/CD Pipeline!

let us discuss the process

  • first create a Dockerfile for the corresponding applications.
  • Then create a Jenkins Node & Jenkins Job.
  • Add the pipeline script that creates the Docker image everytime we made commits to the application.
  • Then add the Dockerhub credintials so, that latest image will be pushed into the central dockerhub everytime.
  • Finally when commits are made the build is automated, latest image is pulled into DOckerhub and latest application can be accessed.
